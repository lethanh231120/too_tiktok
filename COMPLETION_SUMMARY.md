## 🚀 Quick Start (You Can Start Right Now!)

### Step 1: Install Dependencies
```bash
cd /Users/mac/Documents/my_job/tiktok_gen_video_sell
npm install
```

### Step 2: Get API Key
Visit: https://makersuite.google.com/app/apikey
- Click "Create API Key"
- Copy the key

### Step 3: Configure
Edit `.env` file:
```
GEMINI_API_KEY=your_key_here
```

### Step 4: Run
```bash
npm run dev
```

**That's it!** Application opens automatically.
